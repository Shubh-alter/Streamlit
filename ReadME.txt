This Directory is for a Streamlit and LLM beginner.
This Directory mostly helps the user on how Streamlit app is created, and easy application can be built with the help of the components.
Two type of models have been used to create this Web Application 
    1. LLAMA2 - 7B  :   Llama 2 is a collection of pretrained and fine-tuned generative text models 
                        ranging in scale from 7 billion to 70 billion parameters.
    2. GPT2         :   Generative Pre-trained Transformer 2 (GPT-2) is a large language 
                        model by OpenAI and the second in their foundational series of GPT models.
    3. Pexels       :   Pexels is a provider of stock photography and stock footage. It provides the photo closest to the prompt text.
